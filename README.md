```

## Запуск тестов:

```bash
pytest
pytest -v # визуальный режим
pytest -q # тихий режим
```

## Кастомизация запусков:

### Запуск API/UI тестов: (можно добавить еще метов в pytest.ini)
```bash
pytest API
pytest UI
```


## Создание отчетов в Allure:
```bash
pytest --alluredir=allure-results
```
Команда создаст директорию allure-results, сложит в нее отчет о прогоне. Далее эту директорию можно загрузить в AllureTestOps

## Запуск в многопоточном режиме:
```bash
pytest -n8 # запустит тесты в 8 потоков
```
## Авто перезапуск упавших тестов:
```bash
pytest --reruns 2 # запустит тесты и перезапустит упавший еще 2 раза
```
